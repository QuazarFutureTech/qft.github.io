Services we use

Google

Google is a highly reputable data corporation that offers services that are unparallelled in their performance and uses. As a result we use these services to process data which allows us to get work done much faster.

GitHub

GitHub, is a public data hosting service, and archive. With it we are able to push & test updates to our websites, automate our software development workflows, pull issue requests (to fix bugs), and release beta editions of our prepackaged software.

PayPal

PayPal, is a secure payment processing center, which we use for our transactions (even if you don't have an account). As a result we use paypal for most of our daily transactions.
